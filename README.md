# Trabalhos - Disciplina Inteligência Artificial

Os trabalhos presentes neste repositório foram desenvolvidos durante a disciplina de Inteligência Artificial da Universidade Federal de São Carlos (UFSCar), no semestre 2025.2. 

A disciplina teve como objetivo apresentar os principais conceitos, técnicas e aplicações da área, abordando os seguintes conteúdos:

- Busca informada e desinformada
- Algoritmos evolutivos e busca adversarial
- Representação do conhecimento baseada em lógica
- Programação lógica (Prolog) e algoritmos de inferência
- Aprendizado supervisionado e não supervisionado
- Redes neurais artificiais

## Trabalho 1 – O Problema do Caixeiro Viajante

Este trabalho tem como objetivo desenvolver um **algoritmo evolutivo** para solucionar o Problema do Caixeiro Viajante, buscando encontrar a rota de menor custo que permita visitar todas as cidades uma única vez, partindo de uma cidade inicial e retornando a ela ao final do percurso. Para isso, o problema é modelado por meio de indivíduos que representam possíveis rotas, utilizando operadores evolutivos como seleção, cruzamento, mutação e elitismo para explorar o espaço de soluções e encontrar uma aproximação da melhor rota possível para um conjunto de oito cidades com distâncias geradas aleatoriamente.

## Trabalho 2 – Classificação com Árvore de Decisão

Este trabalho tem como objetivo aplicar e avaliar um algoritmo de **aprendizado supervisionado** baseado em árvores de decisão para classificação de dados. A atividade envolve a preparação e transformação de um conjunto de dados, separação entre dados de treinamento e teste, treinamento do modelo utilizando o algoritmo Decision Tree com critério de entropia e visualização da árvore gerada. Além disso, o trabalho realiza a validação do modelo por meio de métricas de desempenho, como matriz de confusão, acurácia, precisão, revocação e medida-F, permitindo analisar a capacidade do algoritmo em classificar corretamente novos exemplos.

## Trabalho 3 – Agrupamento de Dados com K-Means e DBSCAN

Este segundo trabalho tem como objetivo aplicar técnicas de **aprendizado não supervisionado** para identificar padrões e grupos em conjuntos de dados, utilizando os algoritmos K-Means e DBSCAN. A atividade envolve o pré-processamento dos dados, definição do número adequado de agrupamentos por meio do método do cotovelo e análise da qualidade dos clusters utilizando métricas como ARI, AMI e coeficiente de silhueta. Além disso, o algoritmo DBSCAN é utilizado para identificar agrupamentos e possíveis dados considerados como ruído, permitindo comparar diferentes estratégias de agrupamento e avaliar a capacidade dos algoritmos em descobrir estruturas presentes nos dados sem utilizar previamente as classes dos exemplos.
